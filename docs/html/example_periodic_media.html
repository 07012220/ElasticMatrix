Example: Periodic Media

The Medium class overloads the times, mtimes and plus operators to help define large periodic structures.

Initialize a Medium Object

Firstly define a Medium object with the 'unit cell' of the periodic structure.
my_unit_cell = Medium('PMMA', 0.001, 'glass', 0.002)
my_unit_cell =
